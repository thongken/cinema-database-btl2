# tutor support system backend
Backend cho dự án rạp chiếu phim vsecinema


> [!IMPORTANT]
> Khi clone về nhớ chạy lệnh dưới để cài thư viện.
>
> ```
> npm install
> ```
>
> Sau đó chạy lệnh dưới để kết nối database.
>
> ```
> npm run db:pull
> npm run db:generate
> ```
> Chạy dự án.
> ```
> npm run dev
> ```
